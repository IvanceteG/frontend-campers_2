# Imatges dels models

Descarrega 7 fotos de camper vans (pots usar Unsplash, Pexels o Pixabay — totes són gratuïtes) i guarda-les en aquesta carpeta amb aquests noms exactes:

```
hero.jpg                          ← foto gran del Hero
sunlight-cliff-640.jpg            ← Compacta
volkswagen-california.jpg         ← Icònica
ford-transit-custom.jpg           ← Familiar
mercedes-marco-polo.jpg           ← Premium
fiat-ducato-adventure.jpg         ← Aventura
renault-trafic-spacenomad.jpg     ← Urbana
```

## Recomanacions

- **Format:** `.jpg` (millor compressió per a fotos).
- **Mida:** entre 800x600 i 1600x1000 px és suficient. Next.js les optimitzarà.
- **Pes:** sota 500 KB cadascuna després de comprimir.

## On buscar

## I si encara no en tinc?

Cap problema: l'aplicació mostra un **placeholder elegant** mentre no hi ha imatge. Pots anar afegint-les progressivament.
